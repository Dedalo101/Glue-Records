Artist Landing Template – Glue Records

To use for a new artist:
1. Use this template → Create new repo
2. Replace [ARTIST NAME HERE], [INSTA_HANDLE], etc. in index.html
3. Upload real photo to assets/artist-photo.jpg
4. Update music embed TRACK_ID
5. Settings → Pages → Enable on main branch
6. Add custom domain (e.g. artist.club) via Porkbun
